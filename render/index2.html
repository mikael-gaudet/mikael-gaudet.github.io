The computer.

I made this retro computer render in the cycles render engine within blender. I spent a weekend making this and utilized uv unwrapping, Node Based materials, Keylighting, and Modeling using loop cuts and subdivision modifiers.

The sofa.

I creating this sofa in about a week. I utilized the node based materials within the cycles render engine, UV unwrapping. The modeling of this sofa was achieved using mirror modifiers, subdivision modifiers, smoothing, edge split modifier, and loop cuts, and blenders auto verecy welding.

The scifi dining room.

I created this spaceship hallway dining room mostly to explore texturing in blender. To create this render I utilized blenders built in raytracing render engine cycles, using its node based materials (Fernel and diffuse, velvet, glass and glossy nodes), UV unwrapping and texturing and modeling using subdivisions and loop cuts.
